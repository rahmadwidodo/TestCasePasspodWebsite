Software yang saya gunakan yaitu Katalon
Software katalon ini adalah software Automation Tests yang menurut saya mudah digunakan.

Step Automation Test Login Passpoddotcom :
1. Create New Test beri nama projectnya
2. Pada UI Katalon terdapat tombol record berwarna merah, yang berfungsi untuk merecord web mana yang ingin kita test.
3. Kemudian masukkan URL pada kolom URL, disamping kolom URL terdapat icon mozila firefox karena default browser katalon adalah firefox akan tetapi bisa diganti sesuai keinginan.
4. Selanjutnya dengan sendirinya web akan terbuka dan kita melakukan action di website tersebut dengan memasukkan email dan password dan  secara otomatis action yang kita lakukan akan terekam.
5. Kemudian setelah kita selesai melakukan action dan dirasa cukup, kita tinggal menekan tombol stop, dan secara otomatis browser akan tertutup dengan sendirinya. Dan setelah itu kita klik tombol OK kemudian pilih folder repository kemudian OK lagi.
6. Berikan nama testcase sesuai yang diinginkan dan setelah itu klik OK lagi, dan secara otomatis testcase yang kita buat tadi sudah jadi.
7. Langkah selanjutnya kita jalankan test automation tadi, dengan cara klik tombol RUN di atas, nah disini keuntungannya kita bisa memilih browser mana yang ingin kita gunakan.
8. Setelah itu action-action akan secara otomatis melakukan cek sendiri dan terdapat log viewernya untuk melihat apakah testing yang dilakukan berhasil, jika berhasil akan ada informasi bahwa action tersebut PASSED.
